1. Overview

Your goal is to build a small, end-to-end project that demonstrates the three core skills we covered: web scraping to collect data, designing and loading a simple database and writing SQL queries, and creating a minimal web app ("vibe coding") to present your findings.

2. Web Scraping

Your data must come from web scraping (examples: Wikipedia pages, the Mercyhurst Athletics website). In this section describe:

  • Which pages/URLs you scraped and why.
  • The tools you used (for example, Python `requests` + `BeautifulSoup`).

Include a short example showing the HTTP request + parse step you used. For example:

import requests
from bs4 import BeautifulSoup

url = "https://en.wikipedia.org/wiki/Example_page"
resp = requests.get(url, timeout=10)
soup = BeautifulSoup(resp.text, "html.parser")
# find the table or elements, then extract rows
table = soup.find("table", {"class": "wikitable"})
rows = []
for tr in table.find_all("tr")[1:]:
    cols = [td.get_text(strip=True) for td in tr.find_all(["th","td"])]
    rows.append(cols)

If you received help from an AI assistant while writing the scraper, include the prompt you used. Example prompt:

"Write a Python function that uses requests and BeautifulSoup to scrape the 'Results' table from this Wikipedia page: . Return a list of dictionaries with keys: 'date', 'home_team', 'away_team', 'score'. Handle missing values and strip whitespace."

3. Database

Describe your database design. Explain:

  • The table(s) and columns you created.
  • The CSV file(s) you exported from the database for the web app (name and column names).

Include sample SQL queries you used to extract and summarize the data. Example queries:

-- basic counts
SELECT season, COUNT(*) AS matches FROM matches GROUP BY season;

-- top scorers (example)
SELECT player_name, SUM(goals) AS goals FROM player_stats GROUP BY player_name ORDER BY goals DESC LIMIT 10;

-- join example
SELECT m.date, m.home_team, m.away_team, s.player_name
FROM matches m
JOIN player_stats s ON m.match_id = s.match_id
WHERE m.season = '2024';
